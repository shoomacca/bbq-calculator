# Dynamic BBQ & Slow Cook Calculator

**Project type:** TYPE 02 - B2C SaaS
**Framework:** New Genesis

## This Is The Project Root

All planning files for this project are in `.planning/` relative to THIS directory:

```
bbq-calculator/
├── .planning/
│   ├── STATE.md       ← current milestone and shard
│   ├── BRIEF.md       ← project spec
│   ├── STACK.md       ← tech decisions
│   ├── ROADMAP.md     ← all milestones
│   └── milestones/    ← shard files
├── scripts/           ← indiana.js, scribe.js etc
└── CLAUDE.md          ← YOU ARE HERE
```

## Quick Start

```bash
/ng:status      # check where we are
/ng:execute 1.1 # execute current shard
```

## Important

When running ng commands, **always `cd` into this folder first** so relative
paths resolve to this project's `.planning/` — not a parent directory's.
